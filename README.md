# Clase de Muestra

## Objetivos

- Mencionar los beneficios de la nueva tecnología [web components](https://www.webcomponents.org/introduction)
  - Encampsulamiento
  - Reusabilidad
  - Custom Elements
  - Shadow DOM
  - HTML Templates
  - HTML Imports
- Analizar los beneficios de la librería y el JS framework [Polymer](https://www.polymer-project.org/) y las [Progressive Web Apps](https://developers.google.com/web/progressive-web-apps/)
  - Template Based Rendering
  - State Propagation via property binding
  - Declarative Events
  - Basada en standad de nuevas tecnologías web
- Realizar una app de muestra utilizando el `CLI de Polymer` y otros componentes de `Polymer`

## Tareas

- Listado de herramientas a utilizar
  - Terminal de Comandos de Linux
    - `$ mkdir`
    - `$ cd`
    - `$ touch`
  - GIT
    - `git checkout < commit hash >` moverse entre commits
  - [Atom](https://atom.io/) - Editor de Texto
    - [emmet](https://www.emmet.io/)
    - linter
    - linter-js-standard [StandardJS](http://standardjs.com/)
    - linter-htmlhint
    - markdown-preview
  - Google Chrome Browser
    - Developer Tools
- Instalación de librerías y componentes
  - [npm](https://www.npmjs.com/)
    - recomiendo instalar usando nvm
    - `$ npm i -g bower`
    - `$ npm i -g polymer-cli`
  - Polymer CLI
    - `$ polymer init`
    - `$ polymer serve`
  - bower
    - [Polymer Elements](https://www.webcomponents.org)
      - `bower install --save PolymerElements/paper-button`     
      - `bower install --save PolymerVis/mapbox-gl`
